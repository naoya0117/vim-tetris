授業で作った

vimのキー操作で動くテトリスです

<img src="https://github.com/naoya0117/vim-tetris/blob/image/tetris.png?raw=true" width="50%">

## Getting started

```
git clone https://github.com/naoya0117/vim-tetris-plus.git
cd vim-tetris-plus
```
```
bin/run.sh yourname
```
or 
```
bin/run.sh
yourname
```
## uninstall
```
$ cd vim-tetris-plus
$ bin/clear.sh
3
$ cd ..
rm -rf ./vim-tetris-plus
```



## Requirement
- docker
- docker-compose
- (or docker-desktop)
