# vim-tetris
## Description
vimのキー操作で動くテトリスです
![tetris]("https://github.com/naoya0117/vim-tetris/blob/image/tetris.png?raw=true" "tetris")
## Getting Started
```
git clone https://github.com/naoya0117/vim-tetris.git
cd vim-tetris/src
make
./main yourname
```

- wsl+windows-terminal
